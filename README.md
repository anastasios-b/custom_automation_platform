# Προβλήματα Πελάτη

Ο πελάτης μας αντιμετωπίζει ένα πολύ συχνό ζήτημα για την Ελληνική πραγματικότητα:
Λαμβάνει δεδομένα από πολλαπλές πηγές και αναγκάζεται να τα εξάγει και να τα ταξινομεί χειροκίνητα. Αυτό απαιτεί αρκετές ώρες ημερησίως από τον ίδιο ή τον αντίστοιχο εργαζόμενο, οι οποίες θα μπορούσαν να αξιοποιηθούν σε σημαντικότερους τομείς.

# Προτεινόμενη Λύση

Μία πλατφόρμα που:

-   Εξάγει αυτόματα τα δεδομένα από τις διάφορες πηγές
-   Έχει χειροκίνητη λειτουργία για τις τελικές ενέργειες που πρέπει να εκτελέσει
-   Ενημερώνει τον χειριστή για νέα δεδομένα

# Τεχνολογίες

-   Laravel
-   Cloudflare AI Worker (Mistral 7B Parameters)
-   NGINX + PHP-FPM

# Αρχιτεκτονική

1. Admin panel με Blade Views (απλότητα + ταχύτητα development + άμεση σύνδεση με backend)
2. Laravel Scheduler (έλεγχος για ύπαρξη νέων δεδομένων + παρουσίαση ειδοποιήσεων)
3. Cloudflare AI Worker (για λήψη δεδομένων από τις διάφορες πηγές + στατιστική ανάλυση)
4. NGINX + PHP-FPM (proxy)

# Dependencies

-   NGINX
-   PHP-FPM 8.4
-   PHP 8.4
-   Laravel 12.0+
-   Cloudflare AI Worker
